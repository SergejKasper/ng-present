$(function() {
        function showOrHide() {
            var isChecked = $('#ShowFaveNumbers').is(':checked');
            
            // imperative Code dependency
            var fn = $('#FaveNumbers');
            if (isChecked)
                fn.show();
            else
                fn.hide();
        }
        $('#ShowFaveNumbers').change(function() {
            showOrHide();
        });
        showOrHide();
});

    • Umfangreiche Doku und guter Support

    PhoneGap

    • Open-Source-Framework
    • Apps für zehn verschiedene Plattformen entwickeln (iOS, Android, Blackberry…)
    • Bietet zusätzliche Schnittstellen, um Geräte-Hardware wie Kamera oder GPS-Sensor einzubinden
    Open-Source-Framework PhoneGap lassen sich Apps für sieben verschiedene mobile Betriebssysteme entwickeln: iOS, Android, Blackberry, Windows Phone, Palm WebOS, Bada und Symbian. Entwickler schreibt den gesamten Code in HTML, CSS und JavaScript. PhoneGap bietet zusätzliche Schnittstellen, um Geräte-Hardware wie Kameras oder Beschleunigungssensoren einzubinden. Der Zugriff erfolgt dabei durch JavaScript Anschließend kompiliert es alles und stellt Installationsdateien für die verschiedenen Systeme bereit. 2011 von Adobi Systems übernommen seitdem heißt es Apache Cordova

    PhoneGap einrichten

    • Über die Komandozeile (NodeJS vorausgesetzt):
    
                        npm install -g phonegap
    • Manuell über die PhoneGap-Plattform

    PhoneGap benutzen

    • Anlegen eines neuen Projektes:
    
phonegap create meineErsteApp --name "meineErsteApp" --id "de.mms.meineErsteApp"
    • Umwandulung der Web-App für die jeweilige Plattform:
    
                            phonegap run android

    Access Element

    • Erlaubt den Zugriff auf Ressourcen anderer Domains.
    <!-- erlaubt lokale Seiten -->
<access origin="http://127.0.0.1*"/> 
<!-- individueller Zugriff -->
<access origin="https://build.phonegap.com" />
<!-- erlaubt Zugriff auf alle externe Ressourcen -->
<access origin="*" />

    Ripple Emulator

    Ripple Emulator

    • Google Chrome Erweiterung
    • Emulator für multi-Plattformen mobiler Umgebungen
    • Für Entwicklung und Testung von Web/Hybrid-Apps geeignet PhoneGap integriert
    Google Chrome Erweiterung Mit diesem Tool ist es möglich die einzelnen Bildschirmgrößen von Smartphones sowie Events wie dem drücken der Zurücktaste auf einem Androidgerät zu simulieren Nützlich um zu schauen ob das Userinterface für den Großteil von Smartphone- und Tabletgeräten geeignet ist PhoneGap ist in dem Emulator integriert kann so kann simuliert werden wie PhoneGap mit der App interagiert
